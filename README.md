## 🚀 About The Project

DailyFlow is a modern productivity application that helps users:

- ✅ Manage daily tasks
- ✅ Track progress
- ✅ Organize workflow
- ✅ Sync data securely

Built with a scalable architecture:

- Backend: Node.js + Express
- Database: PostgreSQL
- ORM: Sequelize
- Mobile: React Native
- Containerized with Docker
